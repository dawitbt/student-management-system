## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/student-management-system.git
   ```

2. Import database:
   ```bash
   mysql -u username -p database_name < database.sql
   ```

3. Configure settings:
   ```bash
   cp config-sample.php config.php
   nano config.php
   ```
